# 🩸 Blood Group Detection using Fingerprint Analysis

A Machine Learning-based healthcare application that predicts blood groups using fingerprint pattern analysis.

---

# 🚀 Features

✅ Blood Group Prediction  
✅ Fingerprint Pattern Analysis  
✅ Machine Learning Classification  
✅ Streamlit User Interface  
✅ Real-time Prediction  

---

# 🛠️ Tech Stack

- Python
- Machine Learning
- SVM Algorithm
- Streamlit
- Pandas
- NumPy
- Scikit-learn

---

# 📌 Project Overview

This project uses fingerprint characteristics and machine learning algorithms to classify and predict blood groups efficiently.

The application demonstrates how biometric patterns can be analyzed for healthcare-related predictions.

```bash
git clone YOUR_REPOSITORY_LINK
cd blood-group-detection

streamlit run app.py
